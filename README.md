# Conjur::Asset::AWS

Conjur plugin for integrating with AWS.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'conjur-asset-aws'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install conjur-asset-aws
